История создания

Для создания языка JavaScript была нанята компания Netscape, которая хотела разработать язык, позволяющий добавлять интерактивность на веб-страницы.

Брендан Эйх получил задачу внедрить язык программирования в браузер Netscape. Изначально язык назывался Mocha, затем LiveScript. Наконец, он получил своё современное имя — JavaScript.

Интересный факт: JavaScript был выпущен в 1995 году!

Краткий обзор языка JavaScript

JavaScript — это интерпретируемый высокоуровневый язык программирования, используемый в основном для разработки веб-приложений.
